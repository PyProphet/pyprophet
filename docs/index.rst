PyProphet is a Python package designed for scoring and post-processing of DIA-MS data, particularly in the context of OpenSWATH and related workflows. It provides tools for scoring, inference of levels context, and exporting results in various formats. For more information on the general usage of pyprophet, consult the `OpenSWATH website <http://openswath.org/en/latest/>`_.
